1

Captura Automatica de Facturas

Recepcion, extraccion y registro automatizado de facturas PDF

📧
Microsoft Outlook Trigger
Detecta emails con facturas
Configuracion:
• Evento: Nuevo email recibido
• Filtro: hasAttachments = true
• Filtro de remitente para 5 proveedores autorizados
◆
Tiene Adjunto PDF?
Verifica formato
Condicion:
• Verifica que el adjunto sea de tipo PDF
• Rama verdadera: continua el flujo
• Rama falsa: se descarta el email
✔ true
📄
Extract from File
Extrae texto del PDF
Operacion: Extract From PDF
• Lee el contenido textual del archivo PDF adjunto
• Prepara los datos para el procesamiento con IA
🤖
Information Extractor (Claude AI)
Extrae datos estructurados
Datos que extrae:
• N° de Factura
• Nombre del Proveedor
• Monto total
• Fecha de emision
• Fecha de vencimiento
✎
Edit Fields (Set)
Normaliza y estructura datos
Funcion:
• Normaliza formatos de fecha
• Estandariza nombres de proveedores
• Formatea montos correctamente
• Estructura datos para Excel
📊
Microsoft Excel 365
Agrega fila al registro
Operacion: Append (Tabla)
• Recurso: Tabla en OneDrive
• Agrega nueva fila con datos de la factura
• Columnas: N° Factura, Proveedor, Monto, Fechas, Estado
📁
Microsoft SharePoint
Sube PDF a SharePoint
Operacion: Upload (File)
• Recurso: Archivo
• Sube el PDF original a la biblioteca de documentos
• Organiza por carpeta de proveedor
📨
Send Email
Notifica nueva factura
Notificacion:
• Envia email al equipo de finanzas
• Incluye resumen de la factura registrada
• Enlace al archivo en SharePoint
2

Verificacion de Pagos

Conciliacion automatica entre facturas registradas y pagos realizados

⏰
Schedule Trigger
Cada 6 horas
Programacion:
• Se ejecuta cada 6 horas
• Revisa si hay nuevos pagos registrados
📁
Microsoft SharePoint
Consulta facturas pagadas
Operacion: Leer archivos
• Consulta carpeta de facturas pagadas en SharePoint
• Obtiene lista de archivos subidos como comprobantes
📊
Microsoft Excel 365
Lee registro de facturas
Operacion: Get Rows
• Lee todas las facturas con estado "Pendiente"
• Obtiene N° Factura, Proveedor y Monto
💻
Code
Compara facturas vs pagos
Logica:
• Compara facturas registradas con archivos de pago en SharePoint
• Identifica coincidencias por N° Factura o monto
• Marca facturas como "pagadas" si hay match
◆
Factura pagada?
Evalua coincidencia
Condicion:
• true: Se encontro comprobante de pago
• false: Factura sigue pendiente
✔ true
📊
Microsoft Excel 365
Actualiza estado a "Pagada"
Operacion: Update
• Cambia estado de "Pendiente" a "Pagada"
• Registra fecha de pago
• Actualiza columna de observaciones
3

Dashboard y Analisis

Generacion diaria de metricas y comparativas por proveedor

⏰
Schedule Trigger
Diario a las 8:00 AM
Programacion:
• Ejecucion diaria a las 08:00
• Genera metricas actualizadas cada manana
📊
Microsoft Excel 365
Lee facturas (3 meses)
Operacion: Get Rows
• Lee todas las facturas de los ultimos 3 meses
• Incluye facturas pagadas y pendientes
💻
Code
Calcula metricas
Calculos:
• Total facturado por mes
• Cantidad de facturas recibidas
• Promedio por factura
• Comparacion por proveedor
• Porcentaje pagado vs pendiente
📊
Microsoft Excel 365
Actualiza hoja "Dashboard"
Operacion: Update
• Actualiza la hoja "Dashboard" con las metricas calculadas
• Datos listos para graficos en Excel
• Resumen ejecutivo actualizado
